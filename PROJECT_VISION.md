# Finger Maze: 햅틱 경로 탐색 (Haptic Pathfinding)
**주요 목표**: 시각적 정보가 제한된 상황에서 사용자가 오직 햅틱 피드백(진동)과 최소한의 절차적 시각 효과를 통해 보이지 않는 미로를 탈출하는 "눈을 감고 느끼는" 내비게이션 경험을 창조합니다.

## 🌟 컨셉
"손가락이 눈이 된다." 사용자는 지도를 보는 대신, 손가락 끝으로 미로의 벽을 느끼며 길을 찾습니다. 화면은 어둡고 미니멀하며, 사용자의 상호작용이 있을 때만 반응하는 "살아있는" 공간입니다.

## 🛠 핵심 기능
1. **햅틱 벽 감지 (Haptic Wall Detection)**: 손가락이 가상의 벽에 부딪힐 때 강렬한 진동 발생 (Matter.js 기반 물리 엔진 활용).
2. **절차적 빛의 잔상 (Procedural Light Trails)**: 손가락이 움직인 궤적을 따라 미세한 '수묵화' 또는 '빛의 보풀' 느낌의 효과를 주어 탐험한 경로를 시각화 (민호의 알고리즘 적용).
3. **자산 없는 아키텍처 (No-Asset Architecture)**: 100% 코드로 구현되는 비주얼. 이미지 파일(PNG/JPG)을 전혀 사용하지 않아 가볍고 빠름.
4. **적응형 난이도 생성**: Prim's 또는 Recursive Backtracking 알고리즘을 사용하여 매번 새로운 미로를 실시간으로 생성.
